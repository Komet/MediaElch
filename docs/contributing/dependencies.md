# MediaElch's Dependencies

*This document is work-in-progress*

MediaElch has a few dependencies. This document describes what and why
dependencies are used.

 - [Qt][qt]
 - [ffmpeg][ffmpeg]
 - [MediaInfoLib][mediainfolib]
 - [Zenlib][zenlib]
 - [Catch2][catch]
